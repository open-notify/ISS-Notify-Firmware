# ISS-Notify Firmware

This has all the code that gets flashed to the ISS-Notify board.


### Requriements:

    $ sudo apt-get install gcc-avr avr-libc


### Build:

    $ make


### Clean:

    $ make clean


## Documentation

[open-notify.org](http://open-notify.org)
